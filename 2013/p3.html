Second simulation of the power flow

After the first simulation, the next step was to make a second one in order to provide more information of how the power flow works and how the efficiencies of the elements involved on each of the phases affects the global efficiency of the ground station.

This time the input data of the simulation is inside three files.

The first two ones are simple matlab data files that come from the data stored by the ground station and needs to be transformed into the format of the matlab files. One file is for the Force that the wind makes on the kite and reaches the winch of the ground station, and the other one is for the speed.

The third file is also a matlab file that contains the variables used for the efficiencies of the elements of the ground station and can be edited easily in case the components are changed.

This makes an improvement if we compare with the previous simulation, because we can use real data stored by the compact RIO system in the ground station to run a simulation of the power flows on the machine and then we can compare the results.

Also it will be useful to detect where the losses of the system are higher. Detecting these bottlenecks will make easier to try to improve the global efficiency of the system.

About SwissKitePower

The SwissKitePower project is a multidisciplinary research and development effort aimed at creating a totally new device which can harness wind energy at high altitudes. The winds at altitude are stronger and more consistent than those at the ground and represent an enormous potential source of renewable energy.
